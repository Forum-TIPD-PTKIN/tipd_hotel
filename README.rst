=====================
TIPD Hotel Management
=====================



Modul komprehensif untuk Manajemen Hotel yang mencakup seluruh operasional dasar pada hotel, apartemen, atau penginapan.

Fitur utama yang dapat Anda kelola:

* Konfigurasi Properti & Bangunan
* Konfigurasi Kamar dan Fasilitas Hotel
* Manajemen Riwayat Check In dan Check Out
* Pengelolaan Folio (Reservasi Tamu)
* Pembayaran dan Penagihan


Persyaratan Sistem
==================

Modul ini dikembangkan secara khusus untuk berjalan pada **Odoo versi 19.0**. Pastikan server atau *environment* Anda sudah menggunakan Odoo 19 sebelum melakukan instalasi modul ini.
https://www.odoo.com/documentation/19.0/id/administration/on_premise/source.html#
